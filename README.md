Sumuary: When launched, this program is designed to start up and greet the user with an ASCII Chess Baner. So, once the user reaches the main menu screen, they are met with 3 setting options: start new game, settings (still under development) and 3 quite game. To make selections, the user needs to select the number 1 - 3 that corrospond to each option. Currently as of now Start new game and quite work. Settings not functional yet. So if you press 1 and enter, my program will prompt you for your name, 1 for solo mode or 2 for multiplayer mode. Currently, the AI is partially set. So, it will be able to pick its own name. Now the program will ask the user if they want to play as black or white. In solo mode, the user has to play both black and white for the time being since the actual AI is not yet set up to play with an actual player. I'm still tryign to figure out how to wire my AI in. Next the program will ask for difficulty. I have not yet set up any actual difficulty sence that did seem a little challenging. There are 3 levels of difficulty: beginer, intermediate, and advanced. These were just consept ideas. Finally, the program will ask the user if they want to add in music. 1 is for yes. 2 is for no. If no the game begins and the program shows the user the options they choose. If yes, the program asks if they want to loop music or to shuffle their music playist. Eventually the user will load into the game with their chosen music. By default white will always move. My program is designed to tell the user when it is their turn. My program is designed to tell the user if a move is illegal or not. So, this is a partially working program and as of now I am pretty satifed with it.
